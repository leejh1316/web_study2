HTML

Hypertext Markup Language (HTML) is the Standard markup language form for creating web pages and web applications. Web borwsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.
